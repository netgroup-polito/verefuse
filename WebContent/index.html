Verifuse RESTful Web Service

This web service has the following resources

  • verifuse
  • caid
  • sap

1) Verifuse (VERInet FUnction SElection) which is the main resource
2) CAID (CApability IDentifier) which has the task of analyzing policies and selecting the capabilities necessary to satisfy them and to send the capabilities to the ADP module.
3) SAP (Selection And Placement) which has the task to receive the instances of the capability from the ADP module and to select the necessary network functions taking care also to allocate them in the physical server.
